- Un juego en donde el jugador debe hacer click en el circulo en pantalla y este mismo escapa del jugador.
- Este juego se hizo con la finalidad de practicar con la herramienta de GSAP.

## Tecnologias:
- React + Typescript 
- Tailwindcss : Estilos
- GSAP : Animaciones
- Netlify : Deploy

## Caracteristicas
- Juego con sistema de puntajes en base a clicks 
- El circulo escapa del cursor del jugador y reacciona en base a la posicion del mismo.
- Posee ojos que siguen la posicion del cursor del jugador
